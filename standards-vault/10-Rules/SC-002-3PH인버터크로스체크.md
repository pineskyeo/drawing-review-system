# SC-002 3 PHASE 인버터 파츠명 크로스체크

## 원문
> 3 PHASE SCHEMATIC DIAGRAM에 표기된 인버터 파츠명(IVMPA~IVMPL)은 PE BAR 도면과 LAYOUT에서도 동일하게 존재해야 한다.

## 해석 노트
- IVMPA~IVMPL 12개 인버터
- 3 PHASE SCHEMATIC에서 각 인버터 회로 + PE 케이블 확인
- PE BAR (3 PHASE) 도면에서 인버터 접지 연결 확인
- INVERTER PANEL LAYOUT에서 인버터 배치 확인
