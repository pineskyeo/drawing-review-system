# PE-001 PE BAR 케이블 크로스체크

## 원문
> PE BAR 도면에 기재된 PE 케이블(PE##)은 해당하는 SCHEMATIC DIAGRAM 또는 센서 상세도에 동일하게 표기되어야 한다.

## 해석 노트
- PB01-03 (1 PHASE PE BAR): 1 PHASE SCHEMATIC DIAGRAM에서 확인
- PB04 (DC PE BAR): 센서 상세도(TS01, BLB 등)에서 확인
- PB05-06 (AD/SHIELD PE BAR): 센서 상세도(LIT 등)에서 확인
- PB10-13 (3 PHASE PE BAR): 3 PHASE SCHEMATIC DIAGRAM에서 확인
- gN2/CDA 배관접지(PE26 등)는 SCHEMATIC에 미표기 정상 (예외)
- PE100~PE106은 MAIN PB → 하위 BAR 내부 연결 케이블이므로 SCHEMATIC에 미표기 정상 (예외)
